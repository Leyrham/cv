Site web calculatrice

Voici un petit site internet réalisé en cours pour réseaux et web.
Ce site est capable de calculer, de reconnaitre les nombres et d'effectuer des calculs simples tel que des additions, des soustractions, des divisions et des multiplications.
Lors de divisions, si le 2ème nombre est égale à 0, le site demande à l'utilisateur de changer de nombre puisqu'on ne peut pas diviser pas par 0. C'est un projet contenant du php, du css ainsi que bien sur du html. Voici mon code pour cet exercice.
